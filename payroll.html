Preguntas frecuentes

Información inicial para orientarte

¿El servicio es únicamente para empresas grandes?

No. Puede adaptarse a empleadores y organizaciones de distintas dimensiones.

¿Qué información se necesita cada mes?

Novedades de asistencia, licencias, horas, cambios salariales y demás datos relevantes para la nómina.

¿Pueden tomar una liquidación que ya realiza otra persona?

Sí. Primero se revisa la información disponible y se organiza una transición.

¿Incluye asesoramiento jurídico laboral?

El alcance se define al contratar el servicio. Cuando aparece una cuestión jurídica puede analizarse por separado.
